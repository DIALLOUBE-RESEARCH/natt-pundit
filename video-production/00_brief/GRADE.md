# Color grade — CapCut

Avoid default teal-orange (skin-heavy grade; ~80% screen UI + abstract B-roll).

| Zone | Direction |
|------|-----------|
| Shadows | Deep blue-cyan |
| Highlights | Neutral / slightly cool |
| Contrast | Marked, readable UI |
| Consistency | Same LUT on scrcpy + Veo clips |

Optional: import `.cube` LUT into `04_edit/luts/` and apply globally.
