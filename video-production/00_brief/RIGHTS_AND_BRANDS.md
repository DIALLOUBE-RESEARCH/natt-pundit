# Droits & marques — Veo / stock / montage

**Regle:** les clips generes = **ambiance** (humains, stade, Coupe du monde). Pas de pub pour un produit tiers.

## INTERDIT dans les prompts Veo et dans le montage

| Categorie | Exemples interdits |
|-----------|-------------------|
| Telephones marques | iPhone, Samsung Galaxy, logo Apple, notch reconnaissable |
| Sport officiel | Logo FIFA, WC trophy officiel, ecussons federations |
| Maillots signes | Nike swoosh lisible, Adidas stripes + logo, numeros de joueurs celebres |
| Boissons / paris | Coca, FanDuel, Bet365, logos bookmakers |
| Apps concurrentes | Polymarket, DraftKings, etc. |

## AUTORISE

| Element | Comment |
|---------|---------|
| Foule stade | Tribunes pleines, jour, golden hour, documentary |
| Visages fans | Emotion, diversite, maquillage couleurs **generiques** |
| Drapeaux | Couleurs pleines (bleu/blanc/rouge) **sans** ecusson lisible |
| Stade | Architecture generique, pas nom de stade trademark |
| Phone générique | Smartphone générique net et visible dans les mains d'un fan (sans marque, design épuré, écran bord à bord standard) |
| Logos hackathon | Slate post-ambiance : contest + TxODDS (fichiers owner) |

## Clarification « 4s » dans les fichiers

`dur4sec` = **4 secondes** de duree video. **PAS** un modele de telephone. Jamais citer un modele de phone dans un prompt.

## Si Veo genere une marque par accident

→ Ne pas utiliser le clip. Regenerer avec negatives renforces OU couper le plan.
