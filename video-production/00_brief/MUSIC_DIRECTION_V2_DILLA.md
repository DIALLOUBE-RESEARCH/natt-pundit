## 2. ADN J Dilla — traduit en vocabulaire producible

Sources : microtiming studies (Wesleyan "Dilla Feel"), LRB "Dilla Time", MusicRadar/Native Instruments boom-bap guides, Lyria prompt guides Google.

### Ce qui definit le son (a mettre dans les prompts)

| Element Dilla | Description | Mot-cles prompt Lyria |
|---------------|-------------|------------------------|
| **Dilla Time** | Kick en retard, snare legerement rush, hi-hats swing septuplet — friction straight vs swing | `unquantized drums`, `MPC-style swing`, `laid-back behind-the-beat groove`, `human feel not grid-locked` |
| **Boom bap** | Kick "boom" beats 1+3, snare "bap" 2+4, 85–95 BPM | `boom bap hip hop`, `90 BPM`, `dusty drum break`, `muted MPC kick`, `crispy snare` |
| **Samples soul/jazz** | Rhodes, basse chaude, chops mid-phrase (pas loops parfaits) | `warm Fender Rhodes chords`, `soul jazz sample chop feel`, `vinyl warmth`, `lo-fi crunch` |
| **Espace** | Moins-is-plus — place pour VO et UI | `sparse arrangement`, `minimal layers`, `room to breathe` |
| **Texture** | Bitcrush leger, vinyl crackle | `12-bit sampler feel`, `subtle vinyl crackle`, `raw demo mix` |

### Ce qu'il NE faut PAS demander (negative prompt)

`orchestral strings`, `epic trailer`, `EDM drop`, `trap hi-hat rolls`, `four-on-the-floor`, `vocals`, `choir`, `cinematic brass`, `corporate uplifting`, `ukulele`, `acoustic folk`

### Ce qu'on ne peut PAS faire legalement / techniquement

- **Sampler un morceau Dilla** dans la video hackathon → copyright.
- **Lyria 002** : pas d'upload audio reference (instrumental only, text in).
- **Lyria 3** : image-to-music oui ; **audio reference style transfer non** (texte + image seulement).
- Strategie : **"inspired by"** via prompt + **ta piste reference** pour analyse descriptive (BPM, instruments, energie) — pas pour copier.

## 3. Positionnement sonic pour Natt Settlement (foot + paris + agents)

Recherche parallele : docs fintech/sport (Betclic entertainment-first, Dumb Money score electronique, Celtics City orchestral) → pour **app mobile paris WC**, le sweet spot n'est ni pub betting criarde ni orchestre HBO.

### Formule proposee : **"Stadium soul hop"**

```
Boom-bap Dilla groove (human swing)
+ soul/jazz harmonic warmth (Rhodes, basse)
+ tension match-day (montee controlee, pas drop EDM)
+ credibilite produit tech (mix propre, pas lo-fi youtube cheap)
```

| Moment video | Energie musicale | Role |
|--------------|------------------|------|
| Intro Veo 0:00–0:22 | 3–5/10, montee lente | Emotion WC + phone — **pas de gros kick tout de suite** |
| Logos 0:22–0:26 | **silence** (owner) | Respiration avant VO |
| Hook + demo mobile | 4–6/10, groove stable | Head-nod sous Charon (duck -15dB) |
| Pari / wallet | 5–7/10, kick plus present | Stakes sans agressivite casino |
| Agent / MCP | 5/10, plus electronic texture | Shift subtil stem 03 |
| Trust / tests | 3–4/10 ou silence partiel | Clarte chiffres |
| Close CTA | 6/10 swell puis fade | Memorable mais pas cheesy |

**Anti-patterns betting** : pas de sirene casino, pas de "winning horn", pas de tempo 140+ trap, pas de voix chantee.

## 5. Prompts v2 (4 stems — calques sur « You Can't Hold A Torch »)

**Valider puis dire « go lyria v2 ».** Stem 01 seul d'abord si tu veux economiser.

### Stem 01 — `music_01_intro` (0:00–0:26, fade silence logos)

```
Instrumental only. 95 BPM, D sharp major. J Dilla-inspired boom bap for a sports documentary intro.
Warm 70s soul chord loop feel (ethereal, filtered), slowly revealing a dusty MPC drum break with
unquantized swing — kick behind the beat, snare slightly rushed. Subtle vinyl crackle.
Mood: match-day focus, soulful tension, not aggressive.

[0:00-0:08] Filtered soul chords only, soft atmosphere, intensity 2/10.
[0:08-0:12] Muted kick and snare enter, head-nod groove, intensity 4/10.
[0:12-0:18] Full pocket — warm bass, swung hi-hats, intensity 5/10.
[0:18-0:26] Strip drums, chords fade to near silence. Intensity 1/10.
```

### Stem 02 — `music_02_demo` (loop scrcpy — cœur du morceau reference)

```
Instrumental loop only. 95 BPM boom bap hip hop, J Dilla Donuts-era production style.
Warm 70s soul sample chop feel, dusty muffled drum break, MPC unquantized human swing,
muted kick, crispy snare, swung hi-hats, round electric bass. Soulful and confident.
Sparse — room for voiceover. Vinyl warmth, 12-bit sampler texture. Seamless loop ending.
No vocals, no trap, no EDM.
```

### Stem 03 — `music_03_agent` (desktop / MCP — variation plus seche)

```
Instrumental only. Same 95 BPM Dilla pocket as stem 02 but drier mix: slightly less soul pad,
more emphasis on kick-snare swing and subtle analog synth stab. Mood: focus, autonomy, trust.
Boom bap with developer-tool precision. No trap hi-hats, no orchestral strings.
```

### Stem 04 — `music_04_outro` (CTA)

```
Instrumental. 95 BPM soulful boom bap finale, J Dilla inspired. Warm soul chord loop returns
with full drum pocket. [0:00-0:15] steady groove intensity 4/10. [0:15-0:25] gentle lift,
add soft Rhodes layer, intensity 5/10. [0:25-0:30] drums drop, chords resolve and fade to silence.
No vocals.
```

**Negative prompt commun (lyria-002)** :
`vocals, singing, choir, orchestral strings, epic trailer, EDM drop, trap rolls, four-on-the-floor, cymbal crashes, corporate uplifting, ukulele, heavy metal, recognizable samples`
